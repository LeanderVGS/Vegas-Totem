# 🧟‍♂️ Zombie Cleansing Totem

**Your ultimate defense against the undead horde!**

Place the mystical totem anywhere on the map and create a **safe zone** where zombies stand no chance. Whether it's a base, safehouse, or event location — you decide where the undead are pushed back.


## ✨ Features

- **🌍 Place Anywhere** — Deploy the totem at any location on the map
- **📏 Configurable Range** — Set your own radius for zombie removal (adjustable in `client.lua`)
- **⚡ Ultra Performance** — Runs at only **0.01 Resmon** — virtually invisible on your server!
- **🎮 Plug & Play** — Easy installation with no complicated setup
- **🔧 Simple Configuration** — All essential settings centralized in `client.lua`


## 🛠️ Installation

1. Drop the script into your `resources` folder
2. Replace `model_totem.ydr` in `hrs_props/stream/[extra_models]`
3. Ensure the resource is started in your `server.cfg`
4. Adjust settings in `client.lua` to your liking
5. Restart your server — done!


## ⚙️ Configuration

Open `client.lua` and easily tweak the values:

- **Removal Radius** — How large should the safe zone be?
- **Additional Settings** — Customize to fit your needs
